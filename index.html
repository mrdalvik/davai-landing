    # Product Specification: URL Shortener

## Problem
Marketing managers need to track article clicks
with full data control. Bitly/Short.io are cloud-
only, expensive, A/B testing locked behind enterprise.

## MVP Features

### Must Have
1. Custom short aliases (brand.co/launch)
2. Redirect with click recording
3. Click analytics (geo, device, referrer)
4. Auth (email + password)
5. UTM tags auto-appended to destination
6. A/B testing — split traffic by weight

### Won't Have (v1)
- Team features (roles, access)
- Public API
- Custom domains
    # Tech Stack: URL Shortener

## Recommended Stack

Backend:     Go + Chi router
Database:    PostgreSQL + pgx driver
Auth:        bcrypt + JWT (httpOnly cookie)
GeoIP:       MaxMind GeoLite2 (embedded)
Frontend:    React + Vite + shadcn/ui
Deploy:      Docker multi-stage build

## Why Go
  Go + Chi:       perf: 10  deploy: 10
  Node + Express: perf: 6   deploy: 7

Single binary, native speed for redirect
hot path. No Redis needed at MVP scale.

    # Security Requirements

## Threat Model

⚠ Open redirect
  Only http/https, block javascript:
  Validate URL format, max 2048 chars

⚠ Brute-force auth
  Rate limit: 5/min on login per IP
  bcrypt cost ≥ 12, JWT in httpOnly cookie

⚠ Click fraud
  Rate limit: 100/sec on redirect per IP
  Auto-alias: min 6 chars (62^6 = 56B)

⚠ IP privacy (GDPR)
  CLICK_RETENTION_DAYS=90 configurable
