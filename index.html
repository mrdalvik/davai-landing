# Product Specification: URL Shortener

## Problem
Marketing teams need branded short links with
click analytics. Existing tools are expensive
or lack custom alias support.

## MVP Features

### Must Have
1. Custom short aliases (brand.co/launch)
2. Click tracking with geo & referrer
3. Dashboard with real-time analytics
4. User auth (email + password)

### Won't Have (v1)
- Team workspaces
- Public API
- Custom domains
# Tech Stack: URL Shortener

## Recommended Stack

Runtime:      Bun 1.x
Framework:    Hono
Database:     PostgreSQL 16
Cache:        Redis (redirect hot path)
Auth:         JWT + refresh tokens
Frontend:     HTMX + Alpine.js

## Why Bun + Hono
  speed: 9  DX: 9  ecosystem: 7
vs Node+Express:
  speed: 6  DX: 7  ecosystem: 10

Hono's middleware pattern fits the redirect
hot path. Redis caching gives <10ms redirects.

# Security Requirements

## Threat Model

⚠ Open redirect
  Validate destination URLs against
  allowlist, block javascript: URIs

⚠ Brute-force short codes
  Rate limit: 60 req/min per IP
  Minimum code length: 6 chars

⚠ SQL injection
  Parameterized queries only
  No raw SQL string concatenation

⚠ Analytics privacy
  Anonymize IPs, no fingerprinting
